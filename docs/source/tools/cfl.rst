CFL Tools
=========

**csr** – Context-sensitive CFL reachability

**Location**: ``tools/cfl/csr/``

**Usage**:
.. code-block:: bash

   ./build/bin/csr [options] input.bc

**graspan** – Graph analysis framework

**Location**: ``tools/cfl/graspan/``

**Usage**:
.. code-block:: bash

   ./build/bin/graspan [options] input.bc

**pocr** – Partial order constraint resolution tools

**Location**: ``tools/cfl/pocr/``

**Sub-tools**:
* ``pocr-aa`` – Alias analysis variant
* ``pocr-cfl`` – CFL reachability variant
* ``pocr-fp`` – Function pointer analysis
* ``pocr-vf`` – Value flow analysis
