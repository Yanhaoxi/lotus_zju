Analysis Tools
==============

**mcp** – Model Checking Platform

**Location**: ``tools/mcp/mcp.cpp``

**Usage**:
.. code-block:: bash

   ./build/bin/mcp [options] input.bc

**owl** – OWL analysis tool

**Location**: ``tools/owl/owl.cpp``

**Usage**:
.. code-block:: bash

   ./build/bin/owl [options] input.bc

**pdg-query** – Program Dependence Graph query tool

**Location**: ``tools/pdg-query/pdg-query.cpp``

**Usage**:
.. code-block:: bash

   ./build/bin/pdg-query [options] input.bc

**spranalyze** – Sprattus abstract interpretation analyzer

**Location**: ``tools/sprattus/spranalyze.cpp``

**Usage**:
.. code-block:: bash

   ./build/bin/spranalyze [options] input.bc
